# Outputs

## Variantes activas

Usar y mantener estas dos versiones:

1. `Template TP - Macro Portal Transparencia - Version Final Sin QPR Sin Localizacion.xlsm`
   - Reporte final regular.
   - Sin `Periodo de QPR`.
   - Sin `Localizacion`.

2. `Template TP - Macro Portal Transparencia - Export Municipios Applicant Type - Template Aguada.xlsm`
   - Reporte con exportacion automatica individual por municipio.
   - Filtra `Applicant Type = Municipio`.
   - Crea una carpeta y un Excel por municipio.

## Variantes historicas

Las variantes anteriores del proceso estan en `../HIST/` y se conservan solo como referencia.
